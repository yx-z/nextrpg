# PyRPGMaker

A Python-based RPG Maker tool for creating role-playing games.

## Installation

```bash
pip install pyrpgmaker
```

## Usage

```python
from pyrpgmaker import Game

# Create a new game
game = Game()
```

## Features

- Create and manage RPG games
- Customizable characters and maps
- Event system
- Battle system
